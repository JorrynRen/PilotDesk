# PilotDesk 设计规格 v1.0

> 版本: v1.0 | 作者: 简意工作室 (jorryn) | 日期: 2026-05

---

## 1. 设计语言

### 1.1 设计理念

PilotDesk 采用 **简洁、专业、高效** 的设计语言，以深色主题为默认，强调信息层次和操作效率。

### 1.2 品牌标识

- **应用名称**: PilotDesk
- **品牌色**: 蓝紫渐变 `#5B7FFF → #8B5CF6`
- **标志风格**: 科技感、简洁线条、AI 主题

## 3. 布局系统

### 3.1 整体布局

```
┌──────────────────────────────────────┐
│            TitleBar (32px)           │
├────────┬─────────────────┬──────────┤
│        │                 │          │
│Session │   MainPanel     │  Right   │
│ List   │   (会话区域)     │  Panel   │
│(200px) │                 │ (300px)  │
│        │                 │          │
├────────┴─────────────────┴──────────┤
│           StatusBar (28px)           │
└──────────────────────────────────────┘
```

### 3.2 标题栏

- 高度: 32px
- 左侧: 应用图标 + 应用名
- 右侧: 灵感市集按钮 | 侧边栏切换 | 设置按钮 | 分隔 | 窗口操作按钮（最小化、最大化/还原、关闭）
- **整个标题栏可拖动窗口**
- 双击标题栏切换最大化/还原

### 3.3 侧边栏宽度

| 区域 | 宽度 | 可调 |
|------|------|------|
| SessionList | 200px | 否 |
| MainPanel | flex-1 | 自适应 |
| RightPanel | 300px | 可折叠 |

### 3.4 间距规范

| 场景 | 间距 |
|------|------|
| 页面内边距 | 16px |
| 组件间距 | 12px |
| 紧凑间距 | 8px |
| 元素内边距 | 8-12px |

## 4. 组件规格

### 4.1 按钮

| 类型 | 高度 | 圆角 | 内边距 |
|------|------|------|--------|
| 主要按钮 | 32px | 6px | 0 16px |
| 次要按钮 | 32px | 6px | 0 16px |
| 图标按钮 | 28px | 4px | 4px |
| 窗口操作 | 32px | 0 | 4px |

### 4.2 输入框

- 高度: 36px
- 圆角: 6px
- 边框: 1px solid var(--border-default)
- 焦点: border-color → var(--border-active)
- 占位符色: var(--text-muted)

### 4.3 卡片

- 圆角: 8px
- 内边距: 16px
- 背景: var(--bg-secondary)
- 边框: 1px solid var(--border-default)
- 悬停: border-color → var(--border-hover)

### 4.4 消息气泡

| 类型 | 对齐 | 背景 | 圆角 |
|------|------|------|------|
| 用户消息 | 靠右 | 品牌渐变 | 12px 12px 2px 12px |
| Agent 消息 | 靠左 | var(--bg-secondary) | 12px 12px 12px 2px |
| 系统消息 | 居中 | transparent | — |

### 4.5 选项卡

- 高度: 36px
- 激活: 底部 2px 品牌色指示线
- 文字色: 非激活 → var(--text-secondary)，激活 → var(--text-primary)
