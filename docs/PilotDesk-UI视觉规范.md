## 1. 视觉概览

PilotDesk 采用**深色科技风格**，以 `#0F0F14` 为主背景，蓝紫渐变(`#5B7FFF → #8B5CF6`)为品牌色，打造专业的 AI Agent 桌面客户端体验。

---

## 2. 颜色令牌

### 基础色板

```css
:root {
  /* 背景 */
  --bg-primary: #0F0F14;
  --bg-secondary: #1A1A24;
  --bg-tertiary: #242430;
  
  /* 文字 */
  --text-primary: #E8E8ED;
  --text-secondary: #8B8B9E;
  --text-muted: #5A5A6E;
  
  /* 品牌渐变 */
  --brand-start: #5B7FFF;
  --brand-end: #8B5CF6;
  
  /* 功能色 */
  --color-success: #34D399;
  --color-warning: #FBBF24;
  --color-error: #F87171;
  --color-info: #60A5FA;
  
  /* 边框 */
  --border-default: #2A2A3A;
  --border-hover: #3A3A4E;
  --border-active: #5B7FFF;
}
```

## 4. 组件样式详解

### 4.1 标题栏 TitleBar
- **高度**: 32px
- **背景**: var(--bg-primary)
- **下边框**: 1px solid var(--border-default)
- **拖动**: 整个标题栏 `data-tauri-drag-region`
- **右侧按钮**: 灵感市集、侧边栏切换、设置 | 最小化、最大化/还原、关闭
- **窗口操作按钮**: 紧邻窗口右上角，分隔线隔开

### 4.2 会话列表 SessionList
- **宽度**: 200px
- **背景**: var(--bg-primary)
- **右边框**: 1px solid var(--border-default)
- **项高亮**: 左侧 3px 品牌色条
- **项悬停**: var(--bg-secondary)

### 4.3 消息区域 MainPanel
- **背景**: var(--bg-primary)
- **用户气泡**: 渐变背景(#5B7FFF→#8B5CF6)，靠右，圆角 12px 12px 2px 12px
- **Agent气泡**: var(--bg-secondary)，靠左，圆角 12px 12px 12px 2px
- **操作按钮**: 编辑(用户消息)、收藏灵感(用户消息)

### 4.4 右侧面板 RightPanel
- **宽度**: 300px (展开时)
- **背景**: var(--bg-secondary)
- **左边框**: 1px solid var(--border-default)
- **Tab切换**: 灵感 / 技能

### 4.5 灵感卡片
- **背景**: var(--bg-tertiary)
- **圆角**: 8px
- **图标**: 24px，左上角
- **标题**: 14px，font-weight 600
- **内容**: 12px，var(--text-secondary)，最多2行截断
- **图标选择**: 24个图标，grid-cols-12布局（2行完整显示）

### 4.6 API 配置卡片
- **布局**: 列表式
- **操作**: 编辑、删除、测试连接、拖拽排序(六点手柄)
- **测试状态**: 成功/失败横幅 + 延迟ms

### 4.7 设置页选项卡
- **布局**: 水平排列，溢出裁剪(overflow-x-clip)
- **激活指示**: 底部 2px 品牌色线
- **内容区**: flex-1 + overflow-y-auto，不超出容器

## 5. 交互规范

| 交互 | 行为 |
|------|------|
| 标题栏拖动 | 移动窗口位置 |
| 标题栏双击 | 切换最大化/还原 |
| 设置按钮 | toggle 设置页面显示 |
| 会话模式切换 | 下拉菜单向上展开 |
| 灵感点击 | 内容填入会话输入框 |
| 消息编辑 | 内容填充到输入框 |
| API测试 | 卡片内嵌状态横幅 |
| 拖拽排序 | GripVertical 手柄 + @dnd-kit |

## 6. 图标资源

品牌图标存放于项目根目录 `image/` 文件夹：

- `logo.svg` — 应用 Logo（SVG 矢量）
- `icon-16.png` ~ `icon-512.png` — 各尺寸应用图标
- `tray-icon.png` — 系统托盘图标

代码内图标使用 **Lucide React** 图标库。
