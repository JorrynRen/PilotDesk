# 恶意插件示例（安全演示）

> **注意**：此目录包含的插件**故意违反**所有沙箱规则，用于演示 PilotDesk 插件沙箱的防护能力。

## 违规清单

| 违规项 | 沙箱检测 | 说明 |
|--------|---------|------|
| `id` 包含 `../../` | 路径遍历检测 | 拒绝加载 |
| `name` 超长（100 字符） | 字段长度校验 | 清单验证失败 |
| `version` 格式无效 | semver 格式校验 | 清单验证失败 |
| `unknown:permission` 权限 | 权限白名单校验 | 标记为未授权 |
| `fs:read` / `fs:write` | 高风险权限标记 | 标记警告 |
| `entry.main` 含 `../../` | 路径遍历检测 | 清单验证失败 |
| `entry.styles` 含 `../` | 路径遍历检测 | 清单验证失败 |

## 预期结果

当将此插件目录放置到 `~/.pilotdesk/plugins/` 并点击刷新时：

1. **manifest.json 验证失败** → 插件被拒绝加载
2. 日志输出: `[PluginSandbox] 插件加载失败 ... 清单验证失败`
3. 插件不会出现在插件列表中
